# Teltonika_event-reporting-testing
Custom event reporting functionality testing for Teltonika devices
##Automate the testing of Events Reporting
The purpose of this automated test is to create a program that will test all Events Reporting rules variations and trigger these rules to check whether information is sent when it happens and if the sent information is correct.

The assignment must be completed using the Python programming language. Use third version of python. The test must be able to test Events Reporting with every device that has mobile capabilities. All Events Reporting Types and subtypes must be tested. You must use API to configure Events Reporting rules and to trigger it.
